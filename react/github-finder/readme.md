## Usage

### Generate API Key

Create an `.env` file at the root directory and place GitHub API key

```
REACT_APP_GITHUB_URL="https://api.github.com"
REACT_APP_GITHUB_TOKEN="xxxxx xxxxxx xxxxxxx" // API key
```

### Install dependencies

```
npm install
```

### Build application

```
npm run build
```

### Run application

```
npm run start
```
